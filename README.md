# Statistical Analysis Portfolio

A collection of statistical analyses conducted in R, covering common methods used in biomechanics and health sciences research.

---

## Analyses

### Linear Regression
Examines whether psychological factors (pain self-efficacy, pain catastrophizing) predict physical activity levels in individuals with patellofemoral pain.

[View Report](https://timgilgallon.github.io/stats-portfolio/linear_regression_psych_pa.html)

---

### Paired T-Test
Compares single leg hop distance before and after an intervention to assess within-subject changes in performance.

[View Report](https://timgilgallon.github.io/stats-portfolio/paired_ttest_hop_distance.html)

---

### ICC Reliability
Evaluates test-retest reliability of lateral hop distance measurements across two testing sessions using intraclass correlation coefficients.

[View Report](https://timgilgallon.github.io/stats-portfolio/icc_reliability_lateral_hop.html)

---

### Independent T-Test
Compares biomechanical landing angles and hop performance between males and females during lateral hopping tasks.

[View Report](https://timgilgallon.github.io/stats-portfolio/independent_ttest_lateral_hop.html)

---

## Methods

| Analysis | Statistical Test | Key Packages |
|----------|------------------|--------------|
| Linear Regression | `lm()`, Pearson/Spearman | tidyverse, car, stargazer |
| Paired T-Test | `t.test(paired=TRUE)`, Cohen's d | tidyverse, effectsize |
| ICC Reliability | `ICC()` | psych, tidyverse |
| Independent T-Test | `t.test()`, Cohen's d | tidyverse, rstatix, effsize |
